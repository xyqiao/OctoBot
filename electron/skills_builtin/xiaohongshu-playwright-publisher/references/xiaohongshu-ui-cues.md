# 小红书 UI 识别线索

当页面结构变化或固定元素引用不稳定时，优先用这个文件里的可见文案线索做判断。

## 已登录编辑器信号

满足以下至少两项，可判定当前会话已登录：

- URL 包含 `creator.xiaohongshu.com/publish`
- 页面可见文本类似 `发布`、`发布笔记`、`发布图文`
- 存在可编辑的标题或正文输入区域
- 存在图片或视频上传入口

## 未登录信号

出现以下一个或多个强信号，且未出现编辑器信号时，判定为未登录：

- `登录`
- `手机号登录`
- `验证码登录`
- `扫码登录`
- `请先登录`

## 回退导航路径

若直接访问发布页失败，按以下顺序重试：

1. 打开 `https://creator.xiaohongshu.com`
2. 点击可见入口文案，如 `发布`、`创作`、`发布笔记`
3. 每次跳转后重新抓取 `browser_snapshot` 再继续

## 常见中断

- 草稿恢复弹窗
- 风险/安全确认弹窗
- 素材上传处理中状态

先处理中断，再继续原步骤。
